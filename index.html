Limit to only 7 characters

How to Use

Enter your data into the input field above, separating characters by whitespace.

For example: "a 1 b $ c 3 d" or "x Y 9"

Note 1: If you do not separate characters by whitespace, input letters will be regarded as a single character and generated as such.

Note 2: Your input does not have to exceed 7 characters.
